# Responsive Landing Page

Учебный проект для практики верстки адаптивного лендинга.

## 📌 О проекте

Данный проект создан для закрепления навыков:

* HTML5
* SCSS (SASS)
* Адаптивной верстки (mobile-first)
* Работы со сборщиком Parcel

Сайт корректно отображается на:

* мобильных устройствах
* планшетах
* десктопах

---

## 🎯 Цель проекта

Практика верстки и создание адаптивного лендинга с использованием современных инструментов.

---

## 🚀 Технологии

* HTML5
* SCSS (SASS)
* Parcel
* JavaScript (при необходимости)
